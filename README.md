# Rocket Simulation
A simple rocket simulator built with C++, OpenGL, GLFW, GLM, and Dear ImGui. Features a complete solar system simulation with all 8 planets orbiting the Sun using accurate orbital mechanics.

## Features

### Control
- **Keyboard control**
  - `Q`/`E`: Adjustable time scale (Shift+Q/E for faster adjustment)
  - `R`: Reset time scale to 1.0x
  - `W`/`S`: Camera zoom
  - `Space`: Launch/Stop rocket
  - `A`/`D`: Rotate thrust direction
  - `F`: Camera mode -> Free (manually controlled)
  - `L`: Camera mode -> Locked (follow rocket)
  - `1`: Camera mode -> Fixed on Earth
  - `2`: Camera mode -> Fixed on Moon
  - `3`: Camera mode -> Overview (Earth-Moon system)
  - `4`: Camera mode -> Inner Solar System (Mercury to Mars)
  - `5`: Camera mode -> Full Solar System (all 8 planets)
- **Mouse Control**
  - Left mouse button drag: Rotate camera
  - Mouse wheel: Camera zoom

### Solar System Simulation
- **Sun**: Central star at origin (heliocentric coordinate system)
- **8 Planets** with accurate orbital parameters:
  | Planet | Orbit (AU) | Period | Color |
  |--------|-----------|--------|-------|
  | Mercury | 0.387 | 88 days | Gray |
  | Venus | 0.723 | 225 days | Orange |
  | Earth | 1.000 | 365 days | Blue |
  | Mars | 1.524 | 687 days | Red |
  | Jupiter | 5.204 | 11.9 years | Tan |
  | Saturn | 9.583 | 29.5 years | Yellow |
  | Uranus | 19.19 | 84 years | Cyan |
  | Neptune | 30.07 | 165 years | Deep Blue |
- **Moon**: Orbits Earth with 5.145° inclination (white orbit)
- **Orbital inclinations**: Each planet has accurate ecliptic inclination

### Rendering
- Sun, Earth, Moon, and all planets rendering
- Rocket rendering (3D pyramid aligned with velocity)
- Orbital trajectories for all celestial bodies
- Rocket trajectory visualization (real-time flight path)
- Predicted trajectory for rocket
- Map View: Displays celestial body positions in a 2D overview

### Physics Simulation
- **Gravity**: N-body gravitational simulation including Sun and all planets
- **Integration**: Velocity Verlet for accurate orbital mechanics
- **Rocket Physics**: Thrust, fuel consumption, atmospheric drag
- **Time Scale**: Adjustable from 0.1x to 1,000,000x for observing orbital periods

### Simulation Info
The simulation provides real-time data displayed in an ImGui window:

- **Time Scale**: Current simulation speed (displays as x, min/s, hours/s, or days/s)
- **Mass**: Rocket's total mass in kilograms
- **Fuel Mass**: Remaining fuel mass in kilograms
- **Thrust**: Thrust generated by the rocket in newtons
- **Exhaust Velocity**: Exhaust velocity in meters per second
- **Position (Geocentric)**: Rocket's position in geocentric coordinates
- **Velocity**: Rocket's velocity vector
- **Altitude**: Rocket's altitude above Earth's surface in meters
- **Time**: Elapsed simulation time in seconds
- **Launched**: Whether the rocket has been launched
- **Camera Mode**: Current viewing mode

## Install

### Dependencies

- System libraries (Ubuntu/Debian):
    ```bash
    sudo apt-get install libglfw3-dev libglew-dev libglm-dev libgtest-dev libgmock-dev ccache
    ```

- [Dear ImGui](https://github.com/ocornut/imgui): UI rendering
- [nlohmann/json](https://github.com/nlohmann/json): JSON configuration
- [spdlog](https://github.com/gabime/spdlog.git): For processing logs

### Build Instructions

1. Install dependencies: `libglfw3-dev`, `libglew-dev`, `libglm-dev`, etc.
2. Clone the repository and make sure submodules (like imgui) are included.
3. Run build script:
    ```bash
    ./scripts/build.sh
    ```
4. Execute the simulation:
    ```bash
    ./bin/RocketSimulation
    ```

## Verification

Run the orbital mechanics verification script to validate planet parameters:
```bash
python3 verify/verify_orbits.py
```

This verifies:
- Orbital periods using Kepler's third law
- Orbital velocities using v = √(GM/r)
- Orbital distances in AU
